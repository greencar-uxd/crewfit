### 사용 규칙
1. **작업 시작 시**: "현재 작업 중" 표에 본인 세션·작업·시작시각을 추가하고 push.
2. **작업 끝나면**: 해당 줄을 "최근 완료"로 옮기고, 맨 위 **마지막 갱신** 줄을 고친 뒤 push.
3. **새 할 일**: "작업 대기 리스트"에 우선순위와 함께 추가.
4. 충돌을 줄이려면 표 갱신 → 즉시 `git add STATUS.md && git commit && git push`.
